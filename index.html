Välkommen till Musikhjälpen 2022

Om du inte redan har Minecraft på din dator, köp och installera det från Microsoft Store eller från www.minecraft.net

Nedan följer instruktioner för att installera modifikationerna manuellt, det finns även ett enklare sätt med ett program som heter Curseforge, du kan hitta instruktioner och annan information om eventet här: visualsweden.se/mh22

Du kan så klart logga in på servern utan moddar, men kommer då inte kunna se streamen eller röstchatta med de andra deltagarna. Se "Anslut till Servern" här nedanför.

Installera mods

Instruktionerna nedan kan se långa ut, men de är skrivna för att vara så tydliga som möjligt och ge en steg-för-steg instruktion så du förhoppningsvis inte fastnar.

  1. Ladda ner den här .zip.filen: Ladda ner zip
  2. Öppna filen och dubbelklicka på fabric-installer
    Se till att Client är valt uppe i vänster hörn.
    I Minecraft version väljer du 1.19.2
    I Loader version väljer du senast tillgängliga eller 0.14.10
    Klicka sen Install
  3. Efter detta behöver du lägga till modfilerna i Minecraft. Tryck på Windows-knappen längst ner till vänster på ditt tangentbord I sökrutan skriver du %appdata% och sen Enter Dubbelklicka på mappen .minecraft Om det finns en mapp som heter mods så kopierar du över filerna till den, om den saknas så drar du över hela mods-mappen från .zipfilen

Nu har du installerat allt som krävs.

När du startar Minecraft bör du se att det står "fabric-loader-1.19.2" längst ner till vänster. Om det inte gör det kan du klicka på den lilla nedåtpilen för att välja den.

Anslut till Servern

Klicka på Multiplayer, sen på Add Server
Under Server Adress skriver du mh.aroundthecorner.se
Klicka på Done och då ser du servern i listan samt hur många användare som är inloggade.
Dubbelklicka på Servern för att logga in.

Inställningar för tilläggsprogrammen

För att ändra inställningar i modden för livestream: tryck på R. Här kan du ändra volymen genom att dra i slidern och även ändra upplösningen med knappen "Resolution" längst ner i menyn.

För att ställa in modden för Voice Chat trycker du på "V"

  • Knappen med mikrofonen slår av och på din egen mikrofon
  • Knappen med högtalaren gör att du kan stänga av och på ljudet från de andra spelarna

Om du klickar på Settings kommer du till den här skärmen. Här kan du göra olika inställningar för volym och annat.

Observera de två knapparna längst ner där du kan välja vilken mikrofon du ska använda och även vilken högtalare.

En viktig sak är att "Push to Talk" är inställt som Default och du måste då hålla in Caps Lock när du vill prata. Det enklaste är att klicka på den och ändra till "Voice", så känner den automatiskt av när du pratar.

Tips och Felsökning

  • Röst-chatten kommer att vara aktiverad mellan 16-22 samtliga dagar som musikhjälpen sänds på SVT.
  • Om andra användare inte kan höra dig, dubbelkolla att du har rätt mikrofon inställd under "SelectMicrophone", kontrollera även att ikonen för mikrofonen inte är överkryssad.
  • Om din röst är svag eller hackar kan du prova att höja "Microphone Gain", dvs volymen för din mikrofon och sänka "Voice Activation Threshold", dvs hur högt du behöver prata för att mikrofonen ska aktiveras.
  • Om du upplever att streamen laggar eller hackar kan du prova att trycka R och sen välja "Resolution", där kan du ändra till en lägre upplösning.
